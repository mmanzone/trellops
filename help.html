How to use Trellops

This section provides an overview of accessing Trellops and interacting with the completed dashboard.

Initial Setup and Security

Access Trellops by logging in with your Atlassian Trello user account. All configuration information is stored locally on your computer; Trellops does not store user data on its servers. For security, Trellops only requires read permissions from Trello and cannot make any changes to your boards.

Dashboard Interaction and Filtering

  • View Cards: Clicking on a tile displays a detailed list of the cards contained within that Trello list, including any associated labels.
  • Access Trello: Clicking on an individual card in the detailed list will take you directly to that card on the original Trello board.
  • Filtering: The dashboard can be filtered to display cards based on a specific time period (for example, cards created "last week" or "this week").
  • Hide/Show Blocks: Blocks can be temporarily hidden or shown on the final dashboard view using the associated icon.

Setting up Trellops

  1. Selecting a Trello Board

    Choose the specific Trello board for which you wish to create a dashboard. All boards accessible by your Trello account, across all workspaces, will be available for selection.

  2. Creating and Managing Blocks

    Blocks are logical groupings of Trello lists (tiles) on the dashboard.

    • Create Blocks: Define a name for each block (for example, "Backlog", "In progress", "Completed").
    • Reorder: Blocks can be reordered on the dashboard layout at any time.
  3. Configuring List Tiles

    Select Trello lists from your chosen board to display as individual tiles on the dashboard.

    • Assignment: Move to assign each selected list to one of your created blocks.
    • Reorder Tiles: Lists (tiles) can be moved up or down within their assigned block.
    • Colour Customisation: Assign a custom colour to each tile for visual differentiation. If no colour is chosen, the system will assign a random one.
    • Move Menu: If a tile is mistakenly assigned to the wrong block, use the "move" menu to reassign it.
  4. Other Settings
    • Refresh Rate: By default, the dashboard refreshes every minute. This interval can be changed, with a minimum refresh rate of every 30 seconds.
    • Digital Clock: You can enable a digital clock to be displayed on the top left of the dashboard.
    • Theme: The theme (light/dark) will default to following the system settings but can be manually changed.
  5. Advanced Settings

    These settings control how the first card of a list is handled, typically for lists where the first card is a description rather than a task.

    • Do not count the first card in the total: Excludes the first card from the displayed card count.
    • Display the first card as tile description: Allows the content of the first card to be shown as a description on the main dashboard tile, even if it is excluded from the count.
    • Ignore template cards: Does not count template cards from the list in the total on the tile. This is enabled by default so template cards are excluded from the count (reommended).

Export and Import Configuration

Export Layout: To share your dashboard configuration with colleagues: Click on the "More options..." button, then select "Export the configuration to a file." The file will be saved locally.

Import Layout: To apply a shared configuration: Ensure you have access to the same Trello board, then import the shared configuration file. The dashboard will update once the settings are saved.

Trellops: Frequently Asked Questions (FAQ)

Getting Started and Security

Q: How do I access Trellops?
A: You must log in using your existing Atlassian Trello user account.
Q: Is my Trello data stored on your servers?
A: No. All dashboard configuration data is stored locally on your computer. Trellops does not store any user information on external servers.
Q: What permissions does Trellops require for my Trello boards?
A: Trellops only requires read permissions. It cannot create, edit, or delete cards, lists, or boards on your Trello account.
Q: Which Trello boards can I create a dashboard for?
A: You can create a dashboard for any Trello board your user account has access to, across all your workspaces.

Configuration and Display

Q: What is a "Block" in Trellops?
A: A Block is a collection of Trello lists (displayed as tiles) that you group together on the dashboard. This allows for logical separation and the ability to temporarily hide or show large sections of the dashboard.
Q: How do I assign a Trello list to a Block?
A: In the setup, you move to assign the selected Trello lists to the desired Block you have created.
Q: Can I change the order of my Blocks or my Lists?
A: Yes. You can reorder Blocks on the dashboard layout at any time, and you can move Lists (tiles) up or down within their assigned Block.
Q: What happens if I don't assign a color to a tile?
A: The system will automatically assign a random color to the tile.
Q: How often does the dashboard update?
A: By default, the dashboard refreshes every minute. You can adjust this value in the "Other Settings," with a minimum refresh rate of every 30 seconds.

Advanced Settings

Q: Why would the card count on the tile be inaccurate compared to my Trello list?
A: This often happens if the first card in your list is used as a list description or set of instructions, rather than an actual task card.
Q: How do I correct an inaccurate card count?
A: In the "Advanced Settings" for that specific list, select the option: Do not count the first card in the total. This excludes the descriptive card from the final tally.
Q: Can I display the content of that descriptive first card on the tile?
A: Yes. If you want to exclude the card from the count but still use it for context, select the option: Display the first card as tile description.

Sharing and Interaction

Q: How do I view the cards behind a tile?
A: Simply click on the tile. A detailed list of the cards and their associated labels will appear.
Q: How do I go directly to a card in Trello?
A: After clicking the tile to view the detailed card list, click on the individual card you are interested in. This will take you directly to that card on the original Trello board.
Q: Can I share my dashboard layout with my team?
A: Yes. You can Export the configuration to a file using the "More options..." menu. As long as your colleagues have access to the same Trello board and use Trellops, they can import this file.
